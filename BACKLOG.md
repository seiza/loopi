## Dictionnaire (écosystème)
- [ ] **Variable**: A way to store data that can be used and changed throughout a program.
- [ ] **Function**: A reusable block of code that performs a specific task.
- [ ] **Conditional (If/Then)**: A basic logic structure that tells a program to perform different actions depending on whether a condition is true or false.
