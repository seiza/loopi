## Dictionnaire (écosystème)
- [x] **Variable**: A way to store data that can be used and changed throughout a program.
- [x] **Function**: A reusable block of code that performs a specific task.
- [x] **Conditional (If/Then)**: A basic logic structure that tells a program to perform different actions depending on whether a condition is true or false.
- [x] **Loop**: A sequence of instructions that is continually repeated until a certain condition is reached.
