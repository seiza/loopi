**Untrusted** est un jeu d'aventure méta-JavaScript où vous devez modifier le code source du jeu pour atteindre la sortie de chaque niveau.

Présenté comme un roguelike 2D, il offre un éditeur de code où vous pouvez voir la logique régissant le niveau. En modifiant le code, vous pouvez créer des passages, supprimer des obstacles ou même créer de nouveaux objets. C'est un moyen ingénieux et stimulant de pratiquer le JavaScript et la réflexion logique.
