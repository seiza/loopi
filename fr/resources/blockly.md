**Blockly** est une bibliothèque logicielle qui permet d'ajouter un éditeur de code visuel aux applications web et mobiles.

Développé par Google, il utilise des blocs qui s'emboîtent pour représenter les concepts de programmation. C'est le moteur qui alimente de nombreux autres outils d'apprentissage comme Scratch ou Tynker.
