**Thymio** est un petit robot éducatif open-source conçu pour l'apprentissage de la logique et de la programmation.

Il contient de nombreux capteurs (proximité, accéléromètre, température, boutons) et peut être programmé de plusieurs façons :
- **Comportements pré-programmés** : Six modes (amical, explorateur, craintif, etc.) pour découvrir ses capteurs dès la sortie de boîte.
- **VPL (Visual Programming Language)** : Une interface par icônes parfaite pour les plus jeunes.
- **Blockly/Scratch** : Pour les apprenants intermédiaires souhaitant passer aux structures logiques.
- **Aseba (Texte)** : Pour les élèves plus avancés désirant apprendre le code textuel.

Le robot est également compatible avec les briques LEGO, permettant aux enfants de personnaliser son apparence et ses fonctions.
