**TIS-100** est un jeu de réflexion ouvert de Zachtronics dans lequel vous réécrivez des segments de code corrompus pour réparer le TIS-100, un système d'intelligence tessellé fictif.

C'est un jeu de programmation "bas niveau" où vous écrivez du code de type assembleur sur plusieurs nœuds interconnectés. C'est un défi minimaliste et parfait pour ceux qui veulent comprendre le fonctionnement fondamental des ordinateurs.
