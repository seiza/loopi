**Bitsbox** est un coffret d'activités mensuel qui apprend aux enfants à créer leurs propres applications. 

Chaque boîte contient plusieurs projets amusants accompagnés d'instructions étape par étape. Les enfants écrivent du code sur le site de Bitsbox pour construire des applis qu'ils peuvent ensuite utiliser et partager sur de vrais appareils mobiles. 

C'est une excellente façon d'apprendre la syntaxe JavaScript réelle dans un environnement structuré et captivant.
