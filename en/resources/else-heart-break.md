**Else Heart.Break()** is a narrative-driven RPG set in a city where everything is controlled by computers.

You'll gain the ability to "hack" the world using a custom programming language called Språk. This allows you to rewrite the code of any object in the game, from doors to cigarettes. It's a deep, atmospheric experience that encourages experimentation and creativity.
