**Elevator Saga** is a programming game where you write JavaScript to control elevators and transport passengers efficiently.

The challenge is to optimize the elevator logic to handle increasing numbers of passengers across multiple floors. It's a great way to practice JavaScript and learn about optimization and scheduling algorithms.
