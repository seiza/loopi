**TIS-100** is an open-ended puzzle game from Zachtronics in which you rewrite corrupted code segments to repair the TIS-100, a fictional tessellated intelligence system.

It's a "low-level" programming game where you write assembly-like code across multiple interconnected nodes. It's challenging, minimalist, and perfect for those who want to understand how computers work at a fundamental level.
