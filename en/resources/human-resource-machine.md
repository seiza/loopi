**Human Resource Machine** is a visual puzzle game where you use a simple assembly-like language to automate a corporate office worker. 

Each level is a task given by your boss, and you must use logical commands like `INBOX`, `OUTBOX`, `COPYTO`, and `JUMP` to complete it. 

It's a fantastic introduction to low-level programming and logic, challenging you to solve increasingly complex problems with a very limited set of instructions.
