**Shenzhen I/O** is a programming puzzle game by Zachtronics where you are an engineer working for a electronics company in Shenzhen, China.

You are tasked with building circuits using a variety of components from different manufacturers and writing assembly code to make them function according to specifications.

The game is a deep dive into low-level engineering, electronics, and assembly programming, requiring you to read a technical manual to solve complex and rewarding puzzles.
