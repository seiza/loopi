**MHRD** is a hardware design game where you design various hardware circuits using a hardware description language.

Starting with simple logic gates, you'll build increasingly complex components like an ALU, a RAM module, and eventually a fully functional CPU. It's an excellent way to learn about digital logic and computer architecture.
