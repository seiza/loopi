**Code Master** is a logic game (single or cooperative) that teaches fundamental programming concepts without a computer.

The game uses an avatar that must travel through a map to reach a portal. To get there, the child must plan their moves using a sequence of commands.

### The lesson:
It introduces more advanced concepts such as:
- **Loops**: repeating an action.
- **Conditionals**: acting differently depending on the situation.

**Ideal age:** 8+ years old
