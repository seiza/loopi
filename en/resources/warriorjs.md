**WarriorJS** is an open-source game where you teach a warrior how to climb a tower and reach the top floor by writing JavaScript code.

Each level presents new challenges and enemies that require logical thinking and proper use of the JavaScript language. It's a fun and competitive way to learn JavaScript while solving puzzles.
