**Game Builder Garage** is a game for Nintendo Switch that teaches you how to create your own video games.

With the help of funny creatures called "Nodons", you learn the logic of programming step by step. Each Nodon represents a function: movement, buttons, logic (AND/OR), or even 3D objects.

The game offers interactive lessons that guide children (and adults) through the creation of several different games, from a simple platformer to a racing game. It's a great way to understand how a video game is built behind the scenes!
